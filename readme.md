# Centry

Basic html, css & js scaffolding

### Modal

Init:  
`<a href="#" data-toggle="modal" data-target="#mymodal">`

Markup:

```
<div class="modal">
    <div class="modal-dialog">
        <h1>My modal</h1>
    </div>
</div>
```

### Dropdown

Init:

`<a href="#" data-toggle="dropdown">`

Markup:
```
<div class="dropdown">
    <a href="#" data-toggle="dropdown">
    <div class="dropdown-menu">
        <h1>My dropdown</h1>
    </div>
</div>
```
